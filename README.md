# Vapeur
Projet Vapeur

## initialisation
### installer les dépendances
``
npm i 
``
### créé le .env
#### créé le .env à la racine du projet
avec : 
``
DATABASE_URL="file:./database.db
``

## démarrer le serveur

### démarrer prisma
``
npx prisma generate
``
### démarrer le serveur
``
npm start
``
